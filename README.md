# Vend Angular 2

## Install

Clone this repo into new project folder (e.g., `my-proj`):
```shell
git clone https://github.com/rodrigo-delgado/vend-angular-2.git  my-proj
cd my-proj
npm install
```

## Start development server on local host:

```shell
npm start
```
cd client and run:
````shell
npm run tsc
````
Live reloading app at `http://localhost:3000/`


Enjoy!
